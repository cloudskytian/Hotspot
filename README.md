# Hotspot
A tool that automatically re-opens win10 hotspot when the hotspot is closed.

A network connection is required when starting the tool, and no network connection is required after the tool is successfully started.

When you click Enable, if the current hotspot is off, the hotspot will be turned on automatically.

一个在 Win10 热点关闭时自动重新开启热点的小工具。

启动工具时需要有网络连接，成功启动工具后不再需要网络连接。

点击Enable时如果当前热点为关闭状态会自动开启热点。
